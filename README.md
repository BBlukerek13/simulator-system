# C++ Console App with Login and Registration System

A simple, interactive command-line application built in C++ that features a basic user management system (registration and login) and support for custom console commands.

## 🚀 Features

* **User Registration & Login:** Secures the console upon startup.
* **Command System:** Interactive loop with built-in commands:
  * `listcommand` - Shows the list of available commands.
  * `changename` - Allows the user to change their registered name.
  * `changepassword` - Allows the user to change their password.
  * `clear` - Clears the console screen.
  * `stop` - Exits the application.

## 🛠️ Built With

* **Language:** C++
* **IDE:** Visual Studio 2022 / Community

## 💻 How to Run

### Prerequisites
To compile and run this project, you need a C++ compiler (like GCC/MinGW) or an IDE like Visual Studio.

### Steps
1. Clone or download this repository.
2. Open the project folder or the `main.cpp` file in your IDE.
3. Compile and run the code.
4. Follow the on-screen instructions to register and log in.
